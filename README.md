### System Requirements
- Node v23.6.0
- npm 10.9.2

### Deployment
````
git clone git@github.com:johenel/test-plumelo-cocktails-app.git
````
````
cd test-plumelo-cocktails-app
````
````
npm install
````
````
npm start
````
