### System Requirements
- Node v23.6.0
- npm 10.9.2

### Deployment
````
- git clone git@github.com:johenel/test-plumelo-cocktails-app.git
- npm install
- npm start
- Navigate to localhost:5137 
or the localhost url that will show on the terminal
````
